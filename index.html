Arcade Game Project

Tijdens mijn eerste jaar op CMD maakte ik een Flappy Bird-achtige game in Unity met een Arduino als joystickcontroller. Het verhaal: een vogel zoekt appels laag bij de grond en ontwijkt obstakels.

Narratief Project

Bij Media Design 2 maakte ik met mijn groep een video die de emotie obsessie overbracht door middel van beeldtaal. Ik werkte met professionele camera’s, storyboard en sounddesign om de emotie kracht bij te zetten.

Het Provocatieve Object

Voor Discursive Design gaf ik een roos een nieuwe betekenis: een symbool voor de onderdrukking van de LHBTI+ gemeenschap in Amerika. Met Blender paste ik het model aan om mijn verhaal visueel te versterken.
