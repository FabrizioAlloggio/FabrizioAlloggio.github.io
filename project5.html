Mobile Water

Tijdens het vijfde collectief bij mijn CMD studie heb ik samen kunnen werken met Mobile Water. Ik heb hier een opdracht gekregen om te onderzoeken welke markten kansrijk zijn voor RCS. Ik heb een marktonderzoek gedaan en hieruit heb ik voor de game markt gekozen. Ik wilde de vele verschillende kanalen die gamestudio's hebben om hun publiek te bereiken, vervangen door RCS. Na nader onderzoek bleek dat ik ze niet allemaal kon vervangen, maar wel twee door middel van kostenbesparing. Bij dit project heb ik ook conversational RCS flows gemaakt om mijn ideeën duidelijk te maken en goed over te kunnen brengen op de opdrachtgevers.
